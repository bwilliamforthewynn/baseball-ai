# ballpark-ai

A baseball analytics app powered by historical Lahman data, dbt, BigQuery, and Claude.

## Stack
- **BigQuery** — data warehouse
- **dbt** — transformations on Lahman baseball data
- **Superset** — embedded dashboards and charts
- **Clerk** — authentication and user management
- **FastAPI** — backend API
- **Claude API** — natural language chat interface
- **React** — frontend shell
